GUI built from standard components
This example shows how the GUI can be built up from standard components.

The avatar panel is generated automatically, but other buttons and fields are provided using standard components. Components are identified by the class name including the number of the avatar. Multiple instances of some components can be used, but components for the SiGML URL or SiGML Text should occur only once for a given avatar.

To build the GUI, span or div elements can be used. A line break will occur after div elements so if elements are to be on the same line, span should be used.
